# DNS Mega
DNS Mega(abbr. Mega) is a DNS cache.It is based on netfilter and has
many feature and performance enhancements. Mega is a Linux kernel module which has proven to be stable and effective in enviroment for same domain DNS querying.

Mega has been an open source project since now(?). It is being actively developed by engineers from Alibaba Group. Everyone is welcomed to get involved.
